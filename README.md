# mrm-task-ava

[![Build Status](https://travis-ci.org/bdougherty/mrm-task-ava.svg)](https://travis-ci.org/bdougherty/mrm-task-ava)

> [Mrm](https://github.com/sapegin/mrm) task that adds [AVA](https://ava.li).

## What it does

* Updates the npm test script to run AVA
* Installs AVA

## Usage

```
mrm ava
```

## Options

There are no options for this task.
